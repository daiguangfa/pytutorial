如何学习Python¶

你必须通过自学掌握Python的列表中的基础内容，以便你在正式的实验开始后，能够 顺利读懂例子程序，以及编写自己的程序。

为了加速你的学习，我将必须掌握的基本内容的条目归纳出来，方便你对照掌握。 秉承“学以致用”的原则，暂时用不到的东西，我们留给将来再学。而下面提到的 内容，你可以从任何一本 Python 教材中挑出来学习。

因为没有专门的课时讲解 Python 的语法，所以我录制了一批微视频，对重点内 容进行了讲解，希望你在课后自己安排学习。自学的最好办法是照着例子做一遍。 每个视频的长度为 10~15 分钟，讲 1~2 个知识点，希望你每看完一集，都立即 跟着做。每周最好只学3集以内的内容，多注重实践，慢比快有效。

视频课假设你有了一门编程的基础课，比如C语言，所以一般不解释 ABC 的概念。 如果你有疑问，可以在网上百度一下即可。我自己学习 python 是边用边学，边学 边用，看的书是英文版教材，没有看过中文书，如果有些名词我翻的不准确，你 可以悄悄的问我。
